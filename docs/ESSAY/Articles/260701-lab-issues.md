# 什么出门单？

> 260701

先平铺直叙一下吧。2026.7.1 下午，我预约了 Apple 门店 Studio Display 显示器的维修，差不多预估着时间打了车，等车快到时，我把显示器装进箱子，提着坐电梯到了一楼。结果我提着显示器刚过闸机，门口的安保人员就把我拦下，说，老师，带出大件物品是需要出门单的。我说，什么出门单？这个是我自己的显示器，不是实验室的，我要带出去维修。他说，实验室规定大件物品出门是需要出门单的，没有出门单我是不能放行的。我开始生气了，这就是我的东西啊，我包里的电脑也是我的东西啊，为什么不能走？他说，那你需要证明这是你的东西。我直接开骂了，我自己的东西我还需要证明？！你怎么不证明这是实验室的东西呢？！不要拦着我了。我往前跨了一步。好家伙，不愧是保安，训练有素，有力的把我又挡住了。保安来了句，请不要为难我。我无奈的回了句，明明是你在为难我啊。

我其实当时只有孤立无援的感觉，我都不知道要联系谁说明情况。规则就是架在保安头上的刀，他怎么都不肯放我出去。我打的车已经在门口了。我从来没有想到带自己的东西出实验室会被拦住。提类似这样的大件物品出门，这是第四次，前三次都没有这回事儿啊？为什么这次拦我？如果从实验室带东西出门需要出门单，是为了安全考虑，说实话，真的有坏人在背包里面装个 U 盘，比我这种提一个显示器出去的问题大多了，那是不是所有人出实验室都要进行安检了？我今天带自己的物品出门要报备，是不是明天我做所有事情都要报备了？我真的不是一个会生气的人，除非遇到的事情触犯到我的底线。我感受到我的人身自由受到限制，我从来都没有过这样的感受。说实话，我当时差点要报警了，我觉得这根本不讲理，很蛮横。

我知道，保安任何没问题，甚至可以说他很负责，他只是在履行上面制定的规则。我妥协了，把显示器拿回闸机内，按照他说的上了五楼服务台，快速说明了情况。好在，五楼的服务台的人员很快帮我处理好了，有现成的打印好的单子，她知道我比较着急，说我只需要填写物品名称和姓名就可以了，她拍一下物品照片我就可以走了，情况同步在群里了保安很快会看到。回到一楼大厅，保安开始确认群聊消息。我心灰意冷，开始吐起槽来，难怪实验室那么多人离职，不是没有道理啊。说实话，我很喜欢实验室，这是我本科毕业之后一路成长的地方，这里给我提供了很好的科研资源、科研环境。但是今天遇到的这件事情，真的让我心生寒意。

之后我大概梳理清楚是怎么一回事儿了。为了实验室安全着想，制定规则的人设置了针对出门物品需要出门单这件事，规则上明确写着的是「考虑到国有资产的安全管理及不必要的误会，大件物品，无论是实验室资产还是员工私人物品，都需要办理报备...填写出门单」。（我其实事后想到，我当时可以出示显示器的购买记录来证明这是我的私人物品给保安看，不过规则上写的是私人大件物品也要报备，甚至写的是提前半个工作日，哈哈）。出发点肯定是没问题的，但是这根本没法执行。我用来做科研的电脑是我自己买的，实验室提供的电脑质量不够好，我每天都带着这样的个人资产离开办公楼，是不是我每天都要报备啊？电脑比较普遍，不说了。那显示器呢，我觉得实验室的显示器分辨率不够高，不满足我的科研需求，我自费购买了显示器，放在实验室用。如果实验室能把廉价的办公显示器升级为高质量办公显示器，显示器出了问题实验室方面去维修，那我也不会遇到出门被拦住这种事情是吧？说白了，这种规则就是一条废规则，完全是防君子不防小人。真的有人想带点机密的东西，一个 U 盘放在兜里就够了。我只觉得自己不被实验室信任。我觉得更应该是告知提醒员工不要带涉及安全的物品出去，提高员工的责任感，而不是阻拦员工常规的个人行动。晚上我下班发现，门口根本没有保安了，真的想把贵重物品带出实验室，只要别白天走就行了，那这不就是装个样子吗？好吧，那要满足出发点，安全考量，怎么做呢？所有出实验室的物品都过安检，都拍照，都开箱检查是不是重要的实验室资产呗。实验室门禁 24 小时由安保人员看守。说实话，现在这样真的就是做个样子。既然是做样子，大件物品出门，保安拍张照留个档不就行了吗？或者人都到楼下了，在一楼前台登记下，不管是纸质表格登记还是扫码登记，不是更快捷吗？办事的地方放在五楼，检查的地方放在一楼，好好笑啊。有了登记的信息，之后真出了什么事儿，查起来也方便？但这不还是防小人不防君子吗？我要是坏人，我登记就写替罪羊的名字，我把一个东西包装成另一个东西报备好，就说这是我的个人物品，里面藏一堆和实验室安全相关的重要器械，你说这查得到吗？所以啊，这检查要么就做绝，严防严守，现在这种程度根本不够；要么就别做面子工程了，把这条规则取掉，给科研人员和服务人员都省点事儿。

另一点，就假如说这条规则是合理的吧，或者有其他的合理规则——规则是需要提前告知的。我三年前入职，那时的实验室刚刚成立，入职培训极其简陋，规章制度也没有较完整的建立。到如今实验室发展壮大，很多规章制度完善了，但是我从来没有被告知过大件物品出楼是需要报备的，哪怕是自然资产。我到楼下了，才由保安告诉我，说实话我觉得这很滑稽。除了出门单，还有多少规章制度是新定下来但是没有发通知告诉实验室的员工的呢？当员工急着办事儿的时候才被告知，别急，规则写着呢，你得遵守。不是我一个人不知道，之前我记得我也在一楼大厅旁观过什么有人自己买的人体工学椅要带走的时候，保安说要什么出门单。没想到我自己也会经历啊。很巧的是就在昨天，我的表弟说想来参观一下实验室，他对人工智能很感兴趣，虽然他是现在只是高中生，但他个人能力非常强，我觉得说不定未来也会成为实验室的联培生，为科研发展做自己的贡献。我得知他想来，我还在大群里确认了，在他们到大厅之前，两个同学回复我，说只要在前台登记就可以进来了，我自己的印象也是这样。但事实上等他们到达，一楼前台才说私人访客如家人亲属不予接待。我查询过后也确实如此，这条规整制度我觉得多少还是合理的。我只好尴尬的送走了浪费了期待和时间的表弟，觉得是自己没有提前了解规则导致的。但现在想想，其他两位同学也不知道啊，回想今天出楼遇到这种事儿，真的是我的问题吗？我又回想起，之前有个同学带他的同学进来参观，好像也是遇到了这样的事情，他也是到了跟前才知道。这些规章制度，为什么作为实验室重要科研力量的联培生们都不知道？是我们觉悟太低了没有主动去学习实验室散落在各处或者说不知道存放在哪里的规章制度是吗？事实上可能正职员工都不知道吧？这是管理出了问题吧？

最后说下，写下这篇记录的本意还是记录问题（当然也不止这些问题了，OA 流程时间随机不确定，重要的全体会议、集群调整只提前一天甚至几小时通知...），希望这些问题能够得到解决，不过说实话，我一下也想不到能去向谁反馈问题。我作为联培生受了实验室很多恩惠，实验室也可能是之后我会继续拼搏努力贡献的地方，我自然希望实验室能够建设的更好。当然如果无法变得更好，甚至变得更糟糕，我觉得还有很多可以为这个社会做贡献实现人生价值的地方去的，没必要拘泥于这儿。
